## Getting started

Clone the repo

```bash
git clone git@github.com:minhhoccode111/game-management.git
```

Run

```bash
dotnet restore
dotnet run
# or develop with
# dotnet watch run
```

Migration

```bash
dotnet ef migrations add SqlServerMigration
dotnet ef database update
```
